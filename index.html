Presentación, ¿Quien eres tú?

En esta segunda clase, el profesor Niko hizo una actividad de presentación para que él pudiera conocernos.

¿Como fue?

En esta actividad, el profesor trajo un ovillo de lana que se pasaba de persona en persona para dar turno de palabra. Así, se fue formando una cuerda que abrazaba a todos los alumnos y que al final se deshizo uno a uno para poner a prueba si el profesor recordaría el nombre de cada uno de los alumnos.

¿Que se descubrió?

Se descubrió dónde vive cada uno de nosotros, cuántos años tenemos, qué nos gusta hacer y una razón por la cual nos gusta ser latinos, si es que nos gusta.

En esta clase, practicamos mucho la conversasión e escuchamos muchas cosas divertidas. Gracias por la atividad maestro Niko!

Proyecto Web: NikoTeka - Portafolio de las NikoClases

En la clase de hoy, se nos propuso crear un sitio web que sirviera como portafolio para las clases del profesor Niko. Aunque el profesor sugirió usar Wix, decidí hacer un sitio en HTML y CSS para practicar las tecnologías que estamos aprendiendo, especialmente porque usamos Chromebooks y no tengo experiencia con Wix.

Objetivo del Proyecto

El propósito principal de este proyecto es construir una plataforma llamada NikoTeka, que funcionará como una NikoPedia donde se catalogarán todas las NikoClases impartidas por el profesor. Será un recurso accesible para repasar y consultar los contenidos de cada sesión.

Tecnologías y Aprendizajes

Este proyecto nos permite aplicar directamente los conceptos de HTML y CSS vistos en clase, creando estructuras semánticas claras y un diseño visual funcional y atractivo. Es una excelente oportunidad para consolidar conocimientos y ganar experiencia práctica.

Expectativas Futuras

A medida que avancemos con más NikoClases, la NikoTeka crecerá en contenido y funcionalidades, convirtiéndose en una verdadera enciclopedia digital que refleje todo el aprendizaje del curso.
