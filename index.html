Introducción a los Fundamentos de Programación

En esta clase exploramos los conceptos fundamentales de la programación, estableciendo las bases para el desarrollo de software.

Conceptos Clave

Durante la sesión, el profesor Nikollas nos introdujo a los conceptos esenciales como variables, tipos de datos, estructuras de control y la lógica de programación. Aprendimos sobre la importancia de la planificación antes de codificar y cómo estructurar nuestros pensamientos de manera algorítmica.

Los ejercicios prácticos incluyeron la resolución de problemas simples utilizando pseudocódigo, lo que nos ayudó a entender la lógica sin preocuparnos por la sintaxis específica de un lenguaje.

Reflexiones

Esta clase sentó las bases fundamentales para todo el aprendizaje posterior. La metodología del profesor para explicar conceptos abstractos mediante ejemplos concretos fue extremadamente efectiva.

Estructuras de Datos y Algoritmos Básicos

En esta segunda clase profundizamos en las estructuras de datos fundamentales y algoritmos básicos de ordenamiento y búsqueda.

Estructuras Estudiadas

Exploramos arrays, listas enlazadas, pilas y colas. Cada estructura fue analizada en términos de sus ventajas, desventajas y casos de uso más apropiados. El profesor Nikollas utilizó analogías muy efectivas para explicar cada concepto.

Algoritmos de Ordenamiento

Estudiamos algoritmos básicos como bubble sort, selection sort e insertion sort. Aunque no son los más eficientes, nos ayudaron a entender los principios fundamentales del análisis de complejidad.

La práctica incluyó implementaciones manuales de estos algoritmos y análisis de su complejidad temporal y espacial.

Desarrollo Web con HTML y CSS

Esta clase nos introdujo al mundo del desarrollo web, comenzando con los lenguajes fundamentales: HTML y CSS.

HTML - La Estructura

Aprendimos sobre la estructura semántica del HTML5, la importancia de usar las etiquetas correctas para cada tipo de contenido, y cómo crear documentos web bien estructurados y accesibles.

CSS - El Estilo

CSS nos permitió dar vida visual a nuestras estructuras HTML. Exploramos selectores, propiedades, el modelo de caja, y técnicas básicas de layout como flexbox y grid.

Proyecto Práctico

Como culminación de la clase, creamos nuestras primeras páginas web completas, aplicando todo lo aprendido sobre estructura semántica y diseño visual. ¡Incluyendo esta misma NikoTeka!
